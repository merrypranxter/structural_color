# Roadmap

## Current Status (v1.2.0)

### Implemented Features

- [x] Core thin-film interference shader
- [x] 10+ shader effects (biological, optical, mathematical)
- [x] FBM noise library
- [x] Color palette system
- [x] Basic geometry generation
- [x] Post-processing pipeline (VHS, bloom)
- [x] Python analysis tools
- [x] CSV datasets (wavelengths, refractive indices, biological data)
